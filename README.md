# GiftCertificateSystem
The app is designed for interaction with a certificate / tag / user entities
<p>
    <b>JWT auth</b> used along with Spring Security
    <ul>
    <li>Modules: core, model, rest</li>
    <li>Spring Boot, Spring Data, Spring Security</li>
    <li>Tests with embedded Postgres DB</li>
    </ul>

# Business use case

<h3>Actions</h3>
<ul>
<li>JWT generation</li>
<li>User registration</li>
<li>CR User's order</li>
<li>CRUD Certificate (JSON patch update)</li>
<li>Find Certificates by name / description / id / tagName</li>  
<li>Sort Certificates by name / create date (ASC / DESC)</li>    
<li>CRD Tag</li>
</ul>
